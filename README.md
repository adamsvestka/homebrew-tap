# Adamsvestka Tap

## How do I install these formulae?

`brew install adamsvestka/tap/<formula>`

Or `brew tap adamsvestka/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "adamsvestka/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
